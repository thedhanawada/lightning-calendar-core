Lightning Calendar Core - Simple HTML Example

Zero dependencies, pure JavaScript calendar that works anywhere

How Simple Is It?

Just 3 lines of code to get a fully functional calendar:

// 1. Load the library
<script src="lightning-calendar.min.js"></script>

// 2. Create calendar instance
const calendar = new LightningCalendar.Calendar();

// 3. Render it
const renderer = new LightningCalendar.VanillaDOMRenderer(container, calendar);
